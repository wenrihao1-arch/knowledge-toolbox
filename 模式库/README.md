# 模式库

本目录用于保存已经验证过、可复用的 AI 工作模式。

模式不是普通提示词。一个合格模式必须说明：

- 解决什么重复任务。
- 需要什么输入。
- 应该输出什么。
- 什么时候不能用。
- 有哪些常见失败。
- 用哪些用例验证过。

新模式优先按 `模式卡片模板.md` 编写。只有经过至少一组最小评测集验证后，才建议标记为 `active`。

## 推荐状态

| 状态 | 含义 |
|---|---|
| candidate | 候选模式，还没有稳定跑通 |
| active | 当前推荐使用 |
| needs-eval | 已有模式，但缺少验收用例 |
| deprecated | 已被更好模式替代 |
| blocked | 缺少权限、资料或工具，暂时无法执行 |
