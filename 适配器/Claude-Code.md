# Claude Code 适配器

## 接入方式

- 临时使用：要求 Claude Code 读取工具箱入口和对应核心协议。
- 项目级使用：本仓库的 `CLAUDE.md` 统一指向根目录 `AGENTS.md`，由后者维护活动规则。
- 原生接入：可将各核心工具封装为该平台支持的命令或 Skill，具体格式以当前版本为准。

## 动作映射

| 通用动作 | Claude Code 中的执行 |
|---|---|
| `READ` | 使用实际可用的文件或网页读取能力 |
| `SEARCH` | 只有会话确实提供联网能力时执行 |
| `VERIFY` | 打开原始页面并比较独立来源 |
| `WRITE` | 在权限范围内写入环境配置指定目录 |
| `ASK` | 缺少会改变任务方向的信息时询问 |
| `REPORT` | 明确说明联网、读取和写入是否真的完成 |

## Claude Code 注意事项

- 不把“模型知道相关内容”当成已经联网核验。
- 不把搜索摘要当作已经读过原始来源。
- 工具权限和联网能力可能随运行环境变化，必须以当前会话为准。
- 核心协议中的停止条件优先于追求冗长的完整输出。

## 推荐入口规则

```text
工具箱是跨 Agent 协议。
根据用户目标自动路由到对应核心协议，用户不需要说出工具名称；
平台能力不足时按协议降级，不假装执行成功；
归档前读取当前环境配置。
```
